Exercise 6

The website will contain a placeholder for an image of the class/professor using the img tag at the very top of the page with their name or class name under it using a h1-h6 header. The table will be used to provide the students with a schedule of how the classwork will be distributed and around what time along with any links to assignments that will be posted later in the semester using the a tag. The link tag will be used to incorporate an external css file into the website so that there will be a foundation for all of the webpages. The style tag will be used for each professor to decorate their website in a unique way. The ol and ul tags will list out the grade percentages for each part of the class as well as the class rules that students need to follow. The title tag allows for students to easily figure out which tab they need to click for which website by having the teachers/class name as the title. Each professor can use the p tag to introduce the class or themselves as well as provide any information they desire about the class. Using these tags each professor will be able to follow a foundational structure for the school while also allowing each professor to uniquely decorate their website. Since each website will be based off a certain style, the students will not have to spend minutes looking around the website to figure out where everything is located.

  2. h1-h6 tag
  3. table tag
  4. a tag
  5. link tag
